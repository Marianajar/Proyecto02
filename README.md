# Proyecto02
##Este proyecto es una aplicación de cifrado web, para ocultar el contenido de un mensaje, archivo o dato de una persona, 

Específicamente, para poder crear claves y contraseñas para las redes sociales, permitiendo tener mas seguridad en este campo.

con el propósito de proteger la confidencialidad de los datos digitales que se ingresen a través de ella haciendo que la información ingresada se traduzca a una nueva forma para que solo las personas con acceso a la clave del usuario puedan leerlo. De esta forma, hacen que las comunicaciones sean mas seguras, pues, los datos estarán codificados, de forma que otras personas no puedan saber el contenido del mensaje.

Para ello hemos hecho uso del Cifrado Cesar, para sustituir cada letra del abecedario por una letra desplazada un número determinado de posiciones, haciendo que el texto no sea entendible por cualquiera.
